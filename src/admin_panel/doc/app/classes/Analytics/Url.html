get_url_calls_graph(app_id)

This method is to populate the varriable url_hits_graph,slowest_url_graph and time_consuming_url_graph with the respective graphs data. url_hits_graph contains the data for url hits for an application. slowest_url_graph contains the data for slowest urls of an application. time_consuming_url_graph contains the data for top time consuming urls of an application.
